# Special interest groups

Special integerest groups (SIGs) are teams within the AlmaLinux OS community that are focused on specific topics. Each SIG has its own channel on our Mattermost (opens new window) chat server chat.almalinux.org (opens new window).

The following SIGs are already established:

  • Core - The AlmaLinux OS distribution developers and maintainers.
  • Cloud - The AlmaLinux OS cloud and container images maintainers.
