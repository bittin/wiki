# Cloud SIG

The Cloud team is responsible for AlmaLinux OS container and cloud images.

The Cloud SIG uses the SIG/Cloud (opens new window) chat channel for communication.

# Deliverables

  • The AlmaLinux OS Docker/Podman images: AlmaLinux/docker-images (opens new window).
  • The AlmaLinux OS cloud images: AlmaLinux/cloud-images (opens new window).
  • The LXC Distrobuilder patches: AlmaLinux/distrobuilder (opens new window).
  • The LXC CI configuration for AlmaLinux OS: AlmaLinux/lxc-ci (opens new window).
  • The container RootFS build tool: AlmaLinux/ks2rootfs (opens new window).

# Help wanted

  • Vagrant + Parallels Desktop box maintainer.
  • Optimized images for Digital Ocean and other cloud providers.
  • Design and implement an images CI/CD pipeline. Automatically check images for security vulnerabilities.
  • Design and automate test scenarios.
  • Technical writers for working on documentation.
  • DevOps engineers for setting up and maintaing the related infrastructure.
